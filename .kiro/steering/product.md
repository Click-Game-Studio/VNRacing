# VNRacing - Product Overview

VNRacing is a mobile-first racing game built with Unreal Engine 5.4+ featuring Vietnamese cultural integration.

## Core Concept
- Mobile racing game targeting Android and iOS
- VN-Tour Campaign: Race through 5 iconic Vietnamese cities (Hà Nội, TP.HCM, Đà Nẵng, Huế, Hội An)
- 63 total tracks with progressive difficulty
- Vietnamese cultural themes: traditional patterns, landmarks, lanterns

## Key Features
- **Racing**: SimpleCarPhysics for mobile-optimized vehicle dynamics, drift mechanics, nitro boost
- **Progression**: XP system, achievements, city-based campaign unlocks
- **Customization**: Visual and performance car modifications
- **Multiplayer**: Nakama matchmaking + Edgegap dedicated servers
- **Race Modes**: Time Attack, Circuit, Sprint, Elimination

## Performance Targets
- 60 FPS on high-end mobile, 30 FPS on low-end
- <2 GB memory budget
- <100ms network latency for multiplayer
- <5s level load times
