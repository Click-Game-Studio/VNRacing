# GAME SETTINGS

GDD Mô tả các tùy chọn Setting của Game.  
**Change Log:**

- First Version (20250919)

1. # **Concepts** 

1. ## **Goals**

- Màn hình Setting cho phép người chơi tinh chỉnh một số tùy chọn liên quan tới Âm thanh, Hình ảnh và các mặt khác của Game.
