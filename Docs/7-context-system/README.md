# 7. Context System

Hệ thống context và hướng dẫn cho VNRacing.

## Mục đích

Section này chứa các tài liệu hướng dẫn và context:
- Coding instructions và guidelines
- Document templates
- Agent configurations
- Development workflows

## Cấu trúc thư mục

| Thư mục | Nội dung |
|---------|----------|
| `instructions/` | Coding guidelines, review checklists |
| `templates/` | Feature templates, PR templates |
| `agents/` | Agent roles và configurations |
| `workflows/` | Development, review, release workflows |

## Nội dung sẽ được migrate

- `_templates/*` → `templates/`
- Standards-based instructions
