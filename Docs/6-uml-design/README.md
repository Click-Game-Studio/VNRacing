# 6. UML Design

Sơ đồ UML Design cho VNRacing.

## Mục đích

Section này chứa các UML design diagrams:
- User story diagrams
- Sequence diagrams
- Activity diagrams

## Cấu trúc thư mục

| Thư mục | Nội dung |
|---------|----------|
| `user-stories/` | User story flow diagrams |
| `diagrams/` | Sequence diagrams, activity diagrams |

## Nội dung sẽ được migrate

- Sequence diagrams từ feature designs
- Activity diagrams từ feature designs
