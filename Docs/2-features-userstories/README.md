# 2. Features & User Stories

Danh sách tính năng và User Stories cho VNRacing.

## Mục đích

Section này chứa requirements và user stories cho từng feature:
- User stories theo EARS format
- Acceptance criteria
- Feature specifications

## Cấu trúc

Mỗi feature có folder riêng:
```
2-features-userstories/
├── car-physics/
├── car-customization/
├── progression-system/
├── multiplayer/
└── ...
```

## Nội dung sẽ được migrate

- `features/*/requirements/` → `2-features-userstories/[feature]/`
