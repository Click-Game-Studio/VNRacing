# Archive

Lưu trữ tài liệu cũ sau khi migration.

## Mục đích

Thư mục này chứa các tài liệu đã được archive sau quá trình restructure:
- Giữ lại để tham khảo lịch sử
- Không còn được maintain
- Có thể xóa sau khi migration hoàn tất và verified

## Cấu trúc thư mục

| Thư mục | Nội dung |
|---------|----------|
| `_templates/` | Old template files |
| `_reports/` | Old report files |
| `features/` | Old feature folder structure (planning/, testing/, etc.) |
| `assets/` | Non-markdown files (PDF, DOCX, PNG, CSV) |

## Lưu ý

⚠️ **Không chỉnh sửa** các file trong thư mục này.
Chỉ dùng để tham khảo trong quá trình migration.
